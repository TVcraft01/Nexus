# NEXUS
### Yours completely. Zero headache. Grows with you.

Nexus is a personal AI that lives on your devices.
Not in the cloud. Not on someone else's server. On yours.

It starts as a tiny seed — under 100MB, one file, double click, done.
It asks two things: your name and your language.
Everything else it figures out by itself. Then it grows.

## Repository Structure

```
Nexus/
├── seed/          ← installer + first-launch logic
├── core/          ← memory, skill loader, conversation loop
├── skills/        ← local skills (community skills in nexus-skills repo)
└── docs/          ← architecture decisions and specs
```

## Quick Start (coming soon)

1. Download the seed installer
2. Double click
3. Say your name and language
4. Nexus does the rest
